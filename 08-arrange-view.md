# Lesson 08 — The Arrange View

**Duration:** 10 minutes
**Outcome:** You build a section-by-section arrangement and compile it into a single Suno-ready prompt.
**Prerequisites:** Lesson 04 complete.

---

## Context (for the LMS module page)

The Arrange View is where you stop describing a song in paragraphs and start building it section by section, with energy levels, key changes, and production cues attached to each block. When you compile, the app turns your arrangement into a single Suno-ready prompt that respects every choice you made. This is how producers think. The Arrange View finally lets you talk to Suno that way.

## What You Will Learn
- How to switch from Chat View to Arrange View
- How to add and edit section blocks (intro, hook, verse, bridge, outro)
- How to set energy levels per section
- How to add bracketed production cues to each section
- Why the bridge always modulates with a key change
- Why the intro is always instrumental unless you say otherwise
- How to compile the arrangement into a single prompt

---

## Step-by-Step Guide

1. Open any mode and click the ARRANGE view tab at the top of the main panel.
2. Add a new section block. The default is Intro.
3. Set the energy level using the slider (0 low, 100 max).
4. Add bracketed production cues, like [filtered pad, no drums].
5. Add the next section. Default order: Intro, Hook, Verse 1, Hook, Verse 2, Bridge, Hook, Outro.
6. On the Bridge, mark the key change (the app suggests one automatically).
7. Confirm the Intro is instrumental. If you want vocals, override it explicitly.
8. Click Compile. The app turns your arrangement into a single prompt.
9. Copy the compiled prompt and paste into Suno.

## Screen Recording Script

### [00:00 — 00:30] Cold open

**[ON SCREEN]** Songwriting Mode, Arrange View active.

**Narration:**
"Arrange View. This is where you stop describing a song and start building it block by block. Ten minutes."

### [00:30 — 01:30] Switching views

**[ON SCREEN]** Highlight the CHAT, ARRANGE, MOOD BOARD tabs. Click ARRANGE.

**Narration:**
"At the top of every mode you have three view tabs. Chat is where we have been living. Arrange is where we are going now. Click it. Notice the main panel changes. Section blocks instead of a chat input."

### [01:30 — 03:30] Adding sections

**[ON SCREEN]** Click add section. Watch a default Intro block appear. Add Hook. Add Verse 1.

**Narration:**
"Click add section. The first one is always Intro. Add Hook. Add Verse 1. Notice the order matches the Myind Sound song structure. Intro, hook, verse, hook, verse, bridge, hook, outro. The app defaults to this order because it works."

### [03:30 — 05:30] Energy levels

**[ON SCREEN]** Drag the energy slider on each block. Show low intro, mid verse, high hook.

**Narration:**
"Each block has an energy slider. Zero is sparse and low. One hundred is full chaos. I usually keep my intro around twenty, my verses around forty, my hooks around eighty, and my bridge around sixty so the second hook hits even harder."

### [05:30 — 07:00] Production cues

**[ON SCREEN]** Click into a section, type bracketed cues like [filtered pad, no drums], [drop: 808s heavy, hi-hat triplets].

**Narration:**
"Add bracketed cues to each section. This is where the production direction lives. Filtered pad no drums on the intro. Heavy 808s and hi-hat triplets on the hook. Half time beat switch on the bridge. Whatever you would tell a producer in the room, type it in brackets."

### [07:00 — 08:00] Bridge key change

**[ON SCREEN]** Open the Bridge block. Highlight the key change suggestion.

**Narration:**
"The bridge always modulates. The app will suggest a key change automatically. You can override it but I recommend you let it ride. Modulating bridges are why the second hook hits harder. This is one of the rules built into every output Suno Writer makes."

### [08:00 — 09:00] Intro stays instrumental

**[ON SCREEN]** Highlight the Intro block. Show that vocals are off by default.

**Narration:**
"And the intro is always instrumental unless you explicitly turn vocals on. Suno tends to drop vocals into intros when you do not tell it not to. Suno Writer prevents that by default."

### [09:00 — 09:45] Compile

**[ON SCREEN]** Click Compile button. Watch the compiled prompt appear. Copy it.

**Narration:**
"When the arrangement looks right, click Compile. The app turns every block into a single Suno-ready prompt. Copy it. Paste into Suno. Generate. The result respects every choice you just made."

### [09:45 — 10:00] Close

**Narration:**
"Next lesson, the Mood Board view. See you there."
