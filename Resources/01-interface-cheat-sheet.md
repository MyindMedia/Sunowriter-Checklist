# Resource 01 — Suno Writer Interface Cheat Sheet

**Format:** 1-page PDF, US Letter landscape (11 in × 8.5 in)
**Used in:** Lesson 02
**Purpose:** Visual map of every panel and button so a member can name everything at a glance.

---

## Brand Layout Spec

- **Background:** `#1A1A1A` full bleed
- **Title:** Inter Black, `#FDB913`, 48pt, top-left
- **Subtitle:** Inter Medium, `#FFFFFF`, 18pt, under title
- **Body labels:** Inter Bold, `#FFFFFF`, 14pt
- **Callout numbers:** Inter Black, `#FDB913`, 24pt inside a `#FF8C00` outlined circle
- **Connector lines:** `#FF8C00`, 2pt
- **Footer:** Inter Regular, `#FFFFFF`, 10pt — "Myind Sound Music Club / Suno Writer v2.0"

## Numbered Callouts

| # | Zone | One-Line Description |
|---|------|----------------------|
| 1 | Top header (SUNO WRITER v2.0) | Home base, app version label |
| 2 | Mode selector | Songwriting / Instrumental / Sample Maker / Analysis |
| 3 | View tabs | CHAT / ARRANGE / MOOD BOARD inside every mode |
| 4 | Sidebar — New Chat | Start a fresh session |
| 5 | Sidebar — Saved Works | Your library, filterable by mode |
| 6 | Sidebar — Templates | Community library, fork or submit |
| 7 | Sidebar — Conversations | Full chat history with sharing built in |
| 8 | Sidebar — User Profile | Default mode and preferences |
| 9 | Header icons | Flow Visualizer / Help tooltips / Clear buffer |
| 10 | Parameter panel | Complexity knob / Rating toggle / H-Pads |

## Production Notes

- Use a real high-resolution screenshot of the dashboard at 1920×1080, color-corrected to match the dark theme
- All callout numbers must be aligned to the brand 8px grid
- Export as PDF/X-1a, embedded fonts, no transparency
