## The 10 Steps

| # | Step | Mode/View | Action |
|---|------|-----------|--------|
| 1 | Idea Capture | Mood Board View | Generate visual creative direction |
| 2 | Reference | Sample Analysis Mode | Analyze a track that matches the vibe |
| 3 | Save DNA | DNA Profiles | Save the analysis as a reusable profile |
| 4 | Sample Maker | Sample Maker Mode | Generate a drumless chop in the same mood |
| 5 | Build the Beat | Instrumental Mode | Use H-Pads to shape texture |
| 6 | Write the Song | Songwriting Mode | Reference the DNA and Mood Board |
| 7 | Arrange | Arrange View | Lay out sections, set energy, add cues |
| 8 | Compile and Generate | Arrange View → Suno | Compile the prompt and paste into Suno |
| 9 | Clean and Prep | Clean Lyrics + Release Prep | Get the song ready for distribution |
| 10 | Save and Ship | Saved Works + Community | Save final and post in the community |

## Decision Branches (smaller side note on the page)

- **Skip Step 4** if you do not need a sample chop
- **Skip Step 9** if your song is already clean and you do not need release prep
- **Loop back to Step 6** if the Suno result needs more lyric work
- **Loop back to Step 7** if the structure needs adjustment

## Production Notes

- Lay out the 10 steps in a U-shape so all nodes fit on landscape US Letter
- Use the brand 8px spacing grid
- Each step number circle should be visually weighty so the eye reads numbers first, labels second
- Export as PDF/X-1a, embedded fonts
