# Resource 03 — Cliché Blacklist Card

**Format:** 1-page PDF, US Letter portrait (8.5 in × 11 in)
**Used in:** Lesson 04
**Purpose:** The exact phrases Suno Writer refuses to use, and why each one is banned.

---

## Brand Layout Spec

- **Background:** `#1A1A1A` full bleed
- **Title:** Inter Black, `#FDB913`, 56pt, top-center
- **Subtitle:** Inter Medium, `#FFFFFF`, 18pt
- **Banned word boxes:** `#1A1A1A` background with red strikethrough effect, gold border `#FDB913` 2pt
- **Banned word text:** Inter Black, `#FFFFFF`, 28pt with red diagonal strikethrough
- **Reason text:** Inter Regular, `#FFFFFF`, 14pt below each word
- **Footer:** Inter Regular, `#FFFFFF`, 10pt

## The Banned List (verbatim from Lawrence's rules)

| Phrase | Why Banned |
|--------|------------|
| in my chest | Overused trope in modern songwriting |
| pressure makes me fold | Tired metaphor everyone uses |
| flame | Vague imagery, lazy substitute for real description |
| light | Vague imagery, used as filler |
| storm | Cliché metaphor for struggle |
| scars | Cliché metaphor for trauma |
| manifest | Overused unless flipped theologically |

## Footer Note

"Suno Writer enforces this blacklist automatically. If a banned word slips through, type 'rewrite without the cliché in line X' and the app will fix it."
