# Resource 05 — Module 1 Index Handout

**Format:** 1-page PDF, US Letter portrait (8.5 in × 11 in)
**Used in:** Welcome email, LMS module landing page, Lesson 01
**Purpose:** A designed table of contents handout members get the moment they start Module 1.

---

## Brand Layout Spec

- **Background:** `#1A1A1A` full bleed
- **Title:** Inter Black, `#FDB913`, 56pt, top-center
- **Subtitle:** Inter Medium, `#FFFFFF`, 18pt
- **Lesson row:** alternating `#1A1A1A` and `#222222` zebra rows
- **Lesson number:** Inter Black, `#FDB913`, 24pt
- **Lesson title:** Inter Bold, `#FFFFFF`, 16pt
- **Lesson duration:** Inter Regular, `#FF8C00`, 14pt right-aligned
- **CTA box:** `#FDB913` background, `#1A1A1A` text, bottom of page
- **Footer:** Inter Regular, `#FFFFFF`, 10pt
