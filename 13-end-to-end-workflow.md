# Lesson 13 — End-to-End Workflow (Capstone)

**Duration:** 20 minutes
**Outcome:** You ship a finished track using every mode in the Myind Sound workflow, idea to released file.
**Prerequisites:** Lessons 01 to 12 complete.

---

## Context (for the LMS module page)

This is the capstone. Every lesson up to this point taught you one piece of Suno Writer. This lesson connects every piece into a single workflow that takes you from idea to ready-to-release in twenty minutes. Follow it once with this video, then run it on your own as many times as you want. This is the workflow I use on real Myind Sound releases.

## What You Will Learn
- The full Myind Sound workflow, step by step
- How to chain modes together in the right order
- How to use DNA Profiles as creative direction
- How to compile an arrangement and ship a Release Prep
- The decision points where you can branch off

## The Myind Sound Workflow

1. **Idea Capture** — generate a Mood Board for the vibe
2. **Reference** — analyze a track you love in Sample Analysis Mode
3. **Save the DNA** — save the analysis as a DNA Profile
4. **Build the Bed** — Sample Maker Mode for a chop, then Instrumental Mode for the beat
5. **Write the Song** — Songwriting Mode using your DNA Profile
6. **Arrange** — switch to Arrange View, lay out sections, add cues
7. **Compile and Generate** — compile the arrangement and paste into Suno
8. **Clean and Prep** — Clean Lyrics if you need a clean version, then Release Prep
9. **Save and Ship** — save the final, post it in the community, distribute it

---

## Step-by-Step Guide

1. Open the Mood Board view. Type your starting feeling or setting. Generate the board.
2. Open Sample Analysis Mode. Drop in a reference track that matches the vibe.
3. Save the analysis as a DNA Profile. Name it after your project.
4. Open Sample Maker Mode. Generate a drumless chop using the same mood and DNA.
5. Open Instrumental Mode. Generate a beat that complements the Sample Maker chop. Use one or two H-Pads to shape it.
6. Open Songwriting Mode. Reference the DNA Profile and the mood board. Write the prompt for the full song.
7. Switch to Arrange View. Lay out the sections, set energy levels, add bracketed cues, confirm the bridge modulation.
8. Click Compile.
9. Paste the compiled prompt into Suno. Generate.
10. If the lyrics need a clean version, click Clean Lyrics.
11. Click Release Prep. Copy the title, description, and metadata.
12. Save the final to your library. Tag it with the project name.
13. Post it in the community with hashtag sunowriter capstone.
