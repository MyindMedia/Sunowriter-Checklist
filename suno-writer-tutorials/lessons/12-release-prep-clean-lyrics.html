Lesson 12 — Release Prep & Clean Lyrics

Duration: 8 minutes Outcome: You take a finished song through Release Prep and generate a clean version of explicit lyrics. Prerequisites: A finished song from Lesson 04 saved in your library.

Context (for the LMS module page)

A song is not done when Suno renders it. It is done when it is ready to ship. Release Prep gives you a release-ready checklist generated specifically for the song you wrote. Clean Lyrics takes any explicit lyric set and gives you a radio-friendly version with the same energy. Both are built for the moment when you stop being a songwriter and become a release artist.

What You Will Learn

  • How to run Release Prep on any saved work
  • What Release Prep generates and how to use it
  • How to convert explicit lyrics into a clean version
  • How to rate a generation so the app learns your taste over time

Step-by-Step Guide

  1. Open Saved Works and pick a finished song.
  2. Click Release Prep.
  3. Read the generated checklist. It includes title suggestions, metadata fields, hook line for marketing, and platform-ready descriptions.
  4. Copy the parts you need into your distribution platform.
  5. To clean lyrics, open the lyric prompt for any saved work.
  6. Click Clean Lyrics.
  7. The app rewrites the lyrics with radio-safe language while keeping the same flow and meaning.
  8. Copy the clean version into a new Suno generation if you need a clean release.
  9. Rate any generation in the app. Your ratings feed your Taste Profile so future outputs get sharper.
