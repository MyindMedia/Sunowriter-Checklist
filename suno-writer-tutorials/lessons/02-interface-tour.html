Lesson 02 — The Interface Tour

Duration: 8 minutes Outcome: You can name every panel and button in the Suno Writer interface and know what each one does at a high level. Prerequisites: Logged in to Suno Writer.

Context (for the LMS module page)

Before you build anything, you need to know where everything lives. Suno Writer is laid out like a studio rack. Once you see the pattern, every mode feels familiar because they all share the same shell. This lesson is the map you will come back to whenever you forget where a button is.

What You Will Learn

  • The rack metaphor and why the layout works that way
  • The 4 mode buttons and what each one is for
  • The 3 view tabs that live inside every mode
  • The sidebar (New Chat, Saved Works, Templates, Conversations, User Profile)
  • The header icons (Flow Visualizer, Help tooltips, Clear buffer)
  • The parameter panel (Complexity knob, Rating toggle, H-Pads)

Step-by-Step Guide

  1. Log in to Suno Writer.
  2. Look at the top header and find the SUNO WRITER v2.0 label.
  3. Look at the left sidebar. This is your library and account.
  4. Look at the mode selector strip. Click each mode once and watch the main panel change.
  5. Inside any mode, find the view tabs labeled CHAT, ARRANGE, and MOOD BOARD. Click each.
  6. Find the Help toggle in the top right and turn it ON for the first week. Tooltips will appear when you hover over anything.
  7. Find the parameter panel and locate the Complexity knob, the Rating toggle, and the H-Pads.
  8. Open the User Profile in the sidebar and find the Default Mode setting.
  9. Close everything and return to the dashboard.
