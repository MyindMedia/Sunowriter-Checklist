LESSON 04

Songwriting Mode

15 min ▼

GHL Build Steps

Title

Lesson 04 — Songwriting Mode

Description

Generate a full Style Prompt and Lyric Prompt, paste them into Suno, and ship your first song.

Lesson Body

Songwriting Mode is the heart of Suno Writer. You give it a feeling, a topic, or a vibe and it gives you back two separate prompts: a Style Prompt that controls the sound, and a Lyric Prompt that controls the words and structure. Both are formatted exactly how Suno wants them, with no clichés, no artist names, and the song structure that actually works for radio-length tracks.

WHAT YOU WILL LEARN
- When to use Songwriting Mode
- The two outputs and why they live in separate copy blocks
- How to upload a reference track for tempo and key matching
- The Complexity knob in practice (0, 50, 100)
- The Explicit and Clean rating toggle
- The Myind Sound song structure
- The cliché blacklist and why we enforce it
- How to paste both prompts into Suno and generate

WHAT TO DO
1. Click Songwriting Mode and confirm you are in the CHAT view
2. (Optional) Upload a reference track using the upload button
3. Set Complexity to 50 for your first run
4. Set Rating to Explicit or Clean depending on your release
5. Type your idea in plain English
6. Hit send and wait for the response
7. Copy the Style Prompt into the Suno Style field
8. Copy the Lyrics Prompt into the Suno Lyrics field
9. Generate in Suno and listen back
10. Iterate by typing follow-ups in the chat (the app remembers context)
11. When you love it, click Save in the sidebar

HOMEWORK
Generate one full song and post it in the community with hashtag #sunowriter-lesson4

LESSON 05

Instrumental / Remix Mode

12 min ▼

GHL Build Steps

Title

Lesson 05 — Instrumental / Remix Mode

Description

Generate beat-only Suno prompts with no vocals and shape them with the H-Pads.

Lesson Body

Instrumental Mode is for when you want a beat, a remix bed, or a backing track with no vocals at all. The app still gives you two prompts, but the Lyric Prompt becomes pure arrangement direction in brackets. No words. Just production cues, energy levels, and section-by-section instructions.

WHAT YOU WILL LEARN
- When to use Instrumental Mode instead of Songwriting Mode
- How the Lyric Prompt becomes arrangement-only with bracketed cues
- The three H-Pads: Rhythm Builder, Harmonic Shift, Texture/Depth
- How to read and edit the bracketed production direction
- How to use the quick-prompt buttons to start fast

WHAT TO DO
1. Click Instrumental Mode and confirm you are in CHAT view
2. Click a quick-prompt button or type your own idea
3. Hit send and read both prompts
4. Click an H-Pad to shape the next response
5. Send a follow-up like "add a beat switch on the second drop"
6. Copy both prompts and paste into Suno
7. Generate, listen, iterate
8. Save when you love it

HOMEWORK
Build one beat, swap one H-Pad, post both versions in the community.

LESSON 07

Sample Analysis Mode

12 min ▼

GHL Build Steps

Title

Lesson 07 — Sample Analysis Mode

Description

Upload any audio file and walk away with a full breakdown plus a Suno prompt that recreates the vibe.

Lesson Body

Sample Analysis Mode is the most technically powerful feature in Suno Writer. You upload any track and the app runs a six-layer analysis (key, BPM, chord progression, energy map, tonal character, groove, structure) and then layers qualitative listening on top. The output is a full sonic breakdown plus a ready-to-paste Suno prompt that recreates the vibe.

WHAT YOU WILL LEARN
- What gets analyzed and how
- The six-layer pipeline in plain English
- How to use the analysis to recreate vs reference a vibe
- Why artist names get scrubbed and why that protects you
- How to combine Sample Analysis with Songwriting or Instrumental Mode

WHAT TO DO
1. Click Sample Analysis Mode
2. Drag an audio file into the chat area
3. Wait for the analysis to complete (under 30 seconds)
4. Read the full breakdown
5. Scroll to the generated Style Prompt and Lyric Prompt
6. Copy both into Suno to recreate the vibe
7. (Bonus) Switch to Songwriting Mode and use the analysis as creative direction for a brand new song

HOMEWORK
Analyze a song you wish you had produced. Tell us in the community what surprised you about the breakdown.
LESSON 08

The Arrange View

10 min ▼

GHL Build Steps

Title

Lesson 08 — The Arrange View

Description

Build a section-by-section arrangement and compile it into a single Suno-ready prompt.

Lesson Body

The Arrange View is where you stop describing a song in paragraphs and start building it section by section, with energy levels, key changes, and production cues attached to each block. When you compile, the app turns your arrangement into a single Suno-ready prompt that respects every choice you made.

WHAT YOU WILL LEARN
- How to switch from Chat View to Arrange View
- How to add and edit section blocks (intro, hook, verse, bridge, outro)
- How to set energy levels per section
- How to add bracketed production cues to each section
- Why the bridge always modulates with a key change
- Why the intro is always instrumental unless you say otherwise
- How to compile the arrangement into a single prompt

WHAT TO DO
1. Open any mode and click the ARRANGE view tab
2. Add a new section block (default Intro)
3. Set the energy level using the slider
4. Add bracketed production cues
5. Add the next sections in order: Intro, Hook, Verse 1, Hook, Verse 2, Bridge, Hook, Outro
6. On the Bridge, mark the key change
7. Confirm the Intro is instrumental
8. Click Compile
9. Copy the compiled prompt and paste into Suno

DNA Profiles

12 min ▼

GHL Build Steps

Title

Lesson 10 — DNA Profiles (Save, Match, Blend)

Description

Save your sonic DNA and blend profiles to create a signature sound on purpose.

Lesson Body

DNA Profiles are the secret to staying consistent across releases. Every time you analyze a track or generate a song you love, you can save that result as a DNA Profile. The profile captures the sonic fingerprint, not the song itself, so you can reuse it as creative direction forever.

WHAT YOU WILL LEARN
- What a DNA Profile actually contains
- How to save a profile from a track you love
- How to use Match to find your closest sonic neighbor
- How to use Blend to merge two profiles
- How to generate from a blended DNA Profile
- Why this is the secret weapon for consistent releases

WHAT TO DO
1. Open Sample Analysis Mode and analyze a track you love
2. At the bottom of the analysis, click Save as DNA Profile
3. Name it
4. Repeat with a second track
5. Open the DNA panel from the sidebar
6. Click Match on profile one to see closest neighbors
7. Click Blend, pick two profiles, hit Blend
8. Click Use in Songwriting Mode to generate from the blended profile

HOMEWORK
Save three DNA profiles. Post one blend in the community.

LESSON 14

Tips, Shortcuts & Troubleshooting

8 min ▼

GHL Build Steps

Title

Lesson 14 — Tips, Shortcuts & Troubleshooting

Description

Avoid the top ten mistakes new users make and know what to do when something breaks.

Lesson Body

Every new tool has the same problem. The first week feels magical, then you hit a wall. Suno Writer is no exception. This lesson is the wall avoided.

TOP 10 MISTAKES & FIXES
1. Vague prompts → name the genre, mood, setting, and energy in one sentence
2. Cranking Complexity to 100 by default → start at 50, push higher only when the song needs to feel like an event
3. Letting clichés through → trust the blacklist, but if one slips, type "rewrite without the cliché in line X"
4. Ignoring the H-Pads → use them for fast texture changes
5. Mixing Style and Lyric prompts in one paste → always paste them in Suno's separate fields
6. Forgetting to upload a reference track for cohesion across an EP → upload the lead single as silent reference for every other track
7. Ignoring DNA Profiles → save three by the end of your first week
8. Skipping Arrange View → use it for any song with more than two sections you care about
9. Releasing without Release Prep → run it on every track before you upload
10. Working in silence → post your work in the community

TROUBLESHOOTING
- Cannot log in: check spam for the magic link first
- Audio upload fails: check file size (max 25 MB) and format (MP3 or WAV)
- Browser quirks: clear cache, try Chrome or Brave, refresh — your work is auto-saved

If you have been stuck for more than ten minutes, post in the community help channel.
